# 📊 Marketing Funnel & Conversion Analysis Dashboard

🚀 *Task 3 – Future Interns Data Science & Analytics Internship*

---

## 🔍 Project Overview

This project focuses on analyzing marketing funnel data to understand customer conversion behavior and identify where users drop off in the funnel.

A dashboard was created using Python to visualize funnel stages, source performance, and conversion insights to improve business decision-making.

---

## 🎯 Objective

* Analyze marketing funnel performance
* Identify drop-off points in customer journey
* Evaluate source/channel effectiveness
* Generate insights to improve conversion rates

---

## 📁 Dataset Description

The dataset includes:

* 👤 Customer demographic details
* 📍 City and employer information
* 💰 Monthly income and financial details
* 📞 Lead interaction status (Contacted)
* 📊 Loan amount and financial metrics
* 📢 Source of lead acquisition

---

## 📊 Dashboard Features

✔ Funnel Flow Analysis (Leads → Contacted → Converted → High Value)
✔ Source Performance Analysis (Bar Chart)
✔ Source Contribution (Pie Chart)
✔ Income Distribution (Box Plot)
✔ Loan Amount by City Category
✔ Income vs Loan Amount (Scatter Plot)
✔ Contacted vs Not Contacted Analysis
✔ Correlation Heatmap

---

## 💡 Key Insights

* 📉 Significant drop-off observed between lead generation and conversion
* 📢 Certain sources contribute more to high-value customers
* 💰 Higher income customers tend to take larger loan amounts
* ⚠️ Many leads are not contacted, indicating missed opportunities

---

## 💼 Business Recommendations

* Improve follow-up strategies for uncontacted leads
* Focus marketing efforts on high-performing sources
* Target high-income customer segments
* Optimize conversion strategies to reduce drop-offs
